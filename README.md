# 🎮 Pinger - Assemble Your Squad!

Ever wanted to invite your friends to join you in gaming without the group chat chaos? **Pinger** is here to save the day! Send quick game invites and PING your friends to hop into the action with you.

## ✨ What is Pinger?

Pinger is your ultimate gaming sidekick. Create a game lobby, invite your friends, and watch them light up with notifications as you PING them to join the fun. No more lost messages in Discord. No more wondering if your friends saw your invite. Just pure, unadulterated gaming coordination.

## 🚀 Tech Stack

Pinger is built on modern, battle-tested technologies:

- **[TanStack Start](https://tanstack.com/start)** - Full-stack React framework with server-side rendering
- **[React](https://react.dev/)** - UI library for interactive components
- **[TypeScript](https://www.typescriptlang.org/)** - Type-safe development for reliability
- **[shadcn/ui](https://shadcn-ui.com/)** - Beautiful, accessible component library
- **[Discord OAuth](https://discord.com/developers/docs/topics/oauth2)** - Seamless authentication with your Discord account
- **Modern Web APIs** - Real-time notifications and gaming coordination

## 🎯 Features

- 🔐 Sign in with your Discord account
- ⚡ Quick game invites
- 🔔 Real-time notifications (PING system)
- 👥 Auto-sync your Discord friends
- 🎮 Game lobby creation
- 📱 Responsive design for desktop and mobile

## 🛠️ Getting Started

```bash
npm install
npm run dev
```

Start pinging your friends and get the squad together! 🚀
